# iPhone 오프라인 웹툰 뷰어

이 폴더는 iPhone Safari에서 홈 화면에 추가해서 오프라인으로 실행할 수 있는 PWA 버전입니다. 가져온 이미지는 iPhone 내부 브라우저 저장소에 보관해서 다음 실행 때도 목록을 기억합니다.

## 사용 방법

1. 이 폴더를 HTTPS 서버로 iPhone Safari에서 엽니다.
2. Safari 공유 버튼에서 `홈 화면에 추가`를 선택합니다.
3. 앱을 열고 `가져오기`에서 웹툰 폴더를 선택합니다.
4. 저장이 끝나면 목록에 머무릅니다. 원하는 작품과 회차를 선택해서 봅니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다. 예: `webtoon-viewer`
2. 이 폴더 안의 파일들을 저장소 루트에 올립니다.
3. 저장소 `Settings` -> `Pages`에서 `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, 폴더는 `/ (root)`를 선택하고 저장합니다.
5. 배포가 끝나면 `https://사용자명.github.io/webtoon-viewer/` 주소를 iPhone Safari에서 엽니다.

## 폴더 구조

다음 구조를 지원합니다.

```text
보관함/
  작품명/
    001화/
      001.jpg
      002.jpg
    002화/
      001.jpg
```

이미지 확장자는 `png`, `jpg`, `jpeg`, `webp`를 지원합니다.

## 주의

- 다시 가져올 때 이미 저장된 파일은 건너뛰고 새 파일만 추가 저장합니다.
- 이미지가 교체된 작품은 목록에서 작품을 선택한 뒤 `선택한 작품 삭제`를 누르고 다시 가져오면 처음부터 저장됩니다.
- 작품별로 마지막에 보던 회차와 스크롤 위치를 기억합니다.
- iOS는 브라우저 저장소 용량을 자동으로 관리할 수 있습니다. 용량이 큰 보관함은 여러 번에 나누어 가져오는 편이 안정적입니다.
- 서비스 워커 오프라인 캐시는 `file://` 또는 일반 `http://IP주소`로 열면 동작하지 않습니다. iPhone 홈 화면 앱으로 오프라인 실행하려면 HTTPS 주소로 한 번 열어야 합니다.
- 같은 Wi-Fi의 로컬 HTTP 서버는 화면 확인용으로는 쓸 수 있지만, iPhone 오프라인 설치 검증용으로는 부족합니다.
